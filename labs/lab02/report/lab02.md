## 1. Модель SIR

### 1.1. Описание модели

Модель SIR (Susceptible–Infected–Recovered) описывает распространение инфекционного заболевания в замкнутой популяции. Популяция делится на три группы:
- **S** – восприимчивые к инфекции;
- **I** – инфицированные (заразные);
- **R** – выздоровевшие (с иммунитетом).

Основные допущения:
- общая численность \(N = S + I + R\) постоянна;
- однородное перемешивание;
- постоянные параметры передачи и выздоровления;
- отсутствие латентного периода;
- пожизненный иммунитет после выздоровления.

### 1.2. Математическая модель

Система дифференциальных уравнений:

\[
\begin{cases}
\displaystyle \frac{dS}{dt} = -\beta c \frac{I}{N} S,\\[8pt]
\displaystyle \frac{dI}{dt} = \beta c \frac{I}{N} S - \gamma I,\\[8pt]
\displaystyle \frac{dR}{dt} = \gamma I,
\end{cases}
\]

где:
- \(\beta\) – вероятность передачи инфекции при контакте;
- \(c\) – среднее число контактов одного человека в единицу времени;
- \(\gamma\) – скорость выздоровления (\(1/\gamma\) – средняя продолжительность болезни).

Базовое репродуктивное число:

\[
R_0 = \frac{\beta c}{\gamma}.
\]

### 1.3. Параметры модели

Для вычислительного эксперимента были выбраны следующие значения:

| Параметр | Значение | Описание |
|----------|----------|----------|
| \(\beta\) | 0.05 | Вероятность заражения при контакте |
| \(c\)    | 10.0 | Число контактов в день |
| \(\gamma\) | 0.25 | Скорость выздоровления (1/γ = 4 дня) |
| \(S_0\)   | 990  | Начальное число восприимчивых |
| \(I_0\)   | 10   | Начальное число инфицированных |
| \(R_0\)   | 0    | Начальное число выздоровевших |

Расчётное значение \(R_0 = (0.05 \cdot 10) / 0.25 = 2.0\).

### 1.4. Результаты моделирования

После решения системы были получены следующие характеристики:

- Пик эпидемии: \(I_{\max} = 223.5\) человек на 17.8 день.
- Итоговая доля переболевших: \(R(\infty) = 796.8\) (80.5% популяции).
- Порог коллективного иммунитета: \(1 - 1/R_0 = 50.0\%\).

## 2. Модель Лотки–Вольтерры

### 2.1. Описание модели

Классическая модель взаимодействия двух видов: жертв (x) и хищников (y). Предложена независимо Лоткой (1925) и Вольтеррой (1926). Описывает циклические колебания численности.

### 2.2. Математическая модель

\[
\begin{cases}
\displaystyle \frac{dx}{dt} = \alpha x - \beta x y,\\[8pt]
\displaystyle \frac{dy}{dt} = \delta x y - \gamma y,
\end{cases}
\]

где:
- \(\alpha\) – коэффициент размножения жертв;
- \(\beta\) – коэффициент поедания жертв хищниками;
- \(\delta\) – эффективность конверсии биомассы жертв в прирост хищников;
- \(\gamma\) – смертность хищников.

### 2.3. Параметры и начальные условия

Использовались следующие значения:

| Параметр | Значение | Описание |
|----------|----------|----------|
| \(\alpha\) | 0.1 | Рождаемость жертв |
| \(\beta\)  | 0.02 | Скорость поедания |
| \(\delta\) | 0.01 | Конверсия |
| \(\gamma\) | 0.3 | Смертность хищников |
| \(x_0\)    | 40.0 | Начальная численность жертв |
| \(y_0\)    | 9.0  | Начальная численность хищников |

### 2.4. Стационарные точки

- Равновесие для жертв: \(x^* = \gamma / \delta = 30.0\)
- Равновесие для хищников: \(y^* = \alpha / \beta = 5.0\)

### 2.5. Результаты моделирования

#### 2.5.1. Динамика численности

Наблюдаются устойчивые колебания. Пик жертв наступает раньше пика хищников (сдвиг фаз около 6.2 единиц времени). Период колебаний, определённый по спектральному анализу, составляет примерно 22.6 единиц времени.
