Теоретическая часть

Описание модели

Экспоненциальный рост — это процесс, при котором скорость изменения величины пропорциональна её текущему значению. Математически это выражается дифференциальным уравнением:

dudt=αu,\frac{du}{dt} = \alpha u, dtdu​=αu,

где:

  • uuu — текущее значение (популяция, капитал и т.д.);
  • ttt — время;
  • α\alphaα — константа скорости роста (если α>0\alpha > 0α>0 — рост, α<0\alpha < 0α<0 — затухание).

Решение уравнения имеет вид:

u(t)=u0eαt,u(t) = u_0 e^{\alpha t}, u(t)=u0​eαt,

где u0u_0u0​ — начальное значение.

Характеристики

  • Время удвоения — время, за которое величина увеличивается вдвое:

    T2=ln⁡2α.T_2 = \frac{\ln 2}{\alpha}. T2​=αln2​.

  • Модель применима в биологии (рост популяции бактерий), финансах (сложные проценты), эпидемиологии (начальная фаза эпидемии) и др.

Ограничения

В реальности экспоненциальный рост не может продолжаться бесконечно из-за ограниченности ресурсов. Модель является идеализированной и служит основой для более сложных моделей (например, логистической).

Реализация на Julia

Для реализации использовался пакет DifferentialEquations.jl. Код оформлен в виде литератного скрипта 01_exponential_growth_lit.jl, который затем был обработан утилитой tangle.jl для генерации чистого скрипта, Quarto-документа и Jupyter-ноутбука.

Выводы

В ходе выполнения лабораторной работы была успешно реализована и исследована модель экспоненциального роста с использованием языка Julia и современных инструментов научного программирования.

  1. Достижение цели работы

    • Реализована математическая модель – дифференциальное уравнение (\frac{du}{dt} = \alpha u) решено численно с помощью пакета DifferentialEquations.jl. Полученное решение (u(t) = u_0 e^{\alpha t}) полностью совпадает с аналитической формулой, что подтверждает корректность численного метода.
    • Проведён анализ – вычислено время удвоения (T_2 = \ln 2 / \alpha), которое оказалось постоянным ((\approx 2.31) при (\alpha = 0.3)), что является ключевой характеристикой экспоненциального роста.

  2. Освоение инструментов и подходов

    • Организация проекта с DrWatson – создана структура проекта, включающая папки для кода (src), данных (data), скриптов (scripts), графиков (plots) и документации. Это обеспечило воспроизводимость и порядок в работе.
    • Литературное программирование (Literate) – скрипт 01_exponential_growth_lit.jl был написан в стиле, объединяющем код и пояснения. С помощью утилиты tangle.jl из него сгенерированы чистый исполняемый скрипт, документ Quarto (.qmd) и Jupyter notebook (.ipynb). Такой подход позволяет легко создавать отчёты и повторно использовать код.
    • Генерация отчёта в Quarto – настроена интеграция Julia с Quarto через специально созданное ядро Jupyter (julia-project), что позволило включать выполняемый код прямо в отчёт и получать итоговый HTML-документ с графиками и таблицами.

  3. Анализ полученных результатов

    • Численное решение визуализировано – график экспоненты имеет характерный изгиб, соответствующий ускорению роста.
    • Таблица данных (первые 5 строк) демонстрирует начальные значения и подтверждает корректность вычислений.
    • Время удвоения, рассчитанное по формуле, совпадает с наблюдаемым на графике: через примерно 2.3 единицы времени популяция удваивается, что легко проверить по таблице (например, значение (u \approx 2.0) при (t \approx 2.31)).

  4. Выявленные особенности и ограничения модели

    • Модель экспоненциального роста адекватно описывает процессы с неограниченными ресурсами (например, размножение бактерий в питательной среде, начисление сложных процентов).
    • Однако в реальных системах рост всегда ограничен внешними факторами, поэтому экспоненциальная модель применима лишь на начальных этапах. В дальнейшем её можно усложнить до логистической (с учётом ёмкости среды), что является естественным продолжением работы.
